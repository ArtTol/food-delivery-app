# Food Delivery App

## Description
This is a full-stack Food Delivery application with two main pages:
- Shops page
- Shopping cart page

## Technologies Used
- Frontend: TypeScript + React
- Backend: Node.js + Express
- Database: PostgreSQL

##  Features
- View list of shops
- Add items to cart
- Fill customer information
- Submit order
- Orders saved to database

## Level of Accomplishment
✔ Frontend implemented with TypeScript  
✔ Backend built without Firebase  
✔ PostgreSQL database connected  
✔ Shops loaded from database  
✔ Orders saved to database  
✔ Two-page navigation (Shops & Cart)  

## Database
